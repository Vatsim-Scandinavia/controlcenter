# Handover

*Handover* is another project maintained by VATSIM Scandinavia.
It is an alternative to [VATSIM Connect](./vatsim-connect.md) which provides you with the ability to

- [x] **GDPR Complicance**: Centralise user authentication in one location to easily scrub systems of user data.
- [x] **Authorization**: Use Handover to authorize or reject user requests to other websites.
- [x] **OAuth2 compatible**: Integrate Handover instead of VATSIM Connect directly with other services for your division.

!!! tip "See :material-github: GitHub for more"
    Find out more [about VATSCA's Handover project](https://github.com/vatsim-scandinavia/handover) by visiting its repository.
